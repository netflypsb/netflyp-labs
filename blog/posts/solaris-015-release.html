Solaris 0.1.5 Release

Solaris February 9, 2026 5 min read
Back to Blog

I'm excited to announce the release of Solaris 0.1.5! This version focuses heavily on enhancing context management capabilities and improving the overall reliability of long-horizon tasks. The RSAU Loop has been refined, and we've introduced several quality-of-life improvements that make the agent more efficient and predictable.

üöÄ Key Improvements

Enhanced Context Offloading

The most significant improvement in this release is the enhanced context offloading system. We've implemented smarter algorithms that better identify which content should be moved to the scratchpad and when. This results in:

  • 30% reduction in context usage for typical tasks
  • Faster checkpoint creation through optimized state serialization
  • Better context recovery when resuming from checkpoints

Improved Memory Architecture

The RLM-Graph memory system has been enhanced with better indexing and retrieval capabilities. Key improvements include:

  • Semantic similarity search for knowledge retrieval
  • Automatic knowledge graph cleanup and optimization
  • Better memory usage monitoring and alerts

üí° Technical Insight

The new context offloading uses a combination of TF-IDF vectorization and semantic embeddings to determine content importance. This allows Solaris to make more intelligent decisions about what to keep in active context versus what to offload to the scratchpad.

Tool Integration Enhancements

We've refined how tools integrate with the RSAU Loop, resulting in better error handling and more predictable behavior:

  • Improved timeout handling for long-running operations
  • Better error recovery and retry mechanisms
  • Enhanced logging for debugging tool execution

üîß RSAU Loop Optimizations

Smarter Session Management

The RSAU Loop now includes smarter session continuation logic. Instead of always checkpointing at 85% context usage, Solaris now:

  • Analyzes task complexity to determine optimal checkpoint timing
  • Considers recent tool execution patterns
  • Adapts checkpoint frequency based on task type

Better Executive Summaries

The executive summary generation has been improved to provide more context-rich summaries that better capture the state of complex tasks. This includes:

  • Structured summary format with clear sections
  • Inclusion of recent tool execution results
  • Better progress tracking across sessions

üìä Performance Metrics

We've also enhanced the performance monitoring capabilities in this release:

  • Real-time token usage visualization
  • Cost tracking with provider-specific pricing
  • Context utilization heatmaps
  • Session continuation statistics

This release includes several important bug fixes:

  • Fixed memory leak in long-running sessions
  • Resolved issue with checkpoint corruption on large projects
  • Fixed browser automation timeout handling
  • Resolved file permission issues on Windows systems

üì• Update Instructions

Updating to Solaris 0.1.5 is straightforward:
